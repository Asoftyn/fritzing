Installing Fritzing

Please make sure your system satisfies one of these requirements:
PC - XP and up
Mac - OSX 10.4 and up
Linux - a fairly recent linux distro with libc >= 2.6

  1. Choose a link above to start downloading the Fritzing package that's right for you.

  2. Unzip your Fritzing folder somewhere convenient on your hard drive.

    • This may also be a good time for you to create a shortcut to the Fritzing application.
    • A free unzipping program can be found here, should you need it.
    • If you are updating your release of Fritzing, your custom files (parts and bins) are not stored with the application, so they will not be destroyed if you delete the older version of the Fritzing application folder.

  3. To start Fritzing:

    • on Linux: double-click fritzing.sh, or try ./fritzing.sh in your shell window
    • on Windows: double-click fritzing.exe
    • on Mac: double-click the Fritzing application

  4. If you experience problems, please try downloading again. This often helps. If it doesn't, have a look at our forums.

  5. Windows note: if you get an error message about "missing dlls", your system probably requires the additional installation of Microsoft C++ runtime libraries. These can be downloaded here.

  6. Linux note: this binary release has been built and tested under Ubuntu 10.4.2 LTS. Distros for other flavors of linux can be found here, and here, and search our forum for even more. We heartily thank openSUSE, Thomas Zimmermann, Debian.org, and other people and organizations for making these available. But please be aware we have not tested these builds.
