Installing Fritzing

First, please make sure your system satisfies one of these requirements:
PC - XP and up
Mac - OSX 10.3 and up
Linux - a fairly recent linux distro with libc >= 2.6 
The current fritzing version has been tested on (K)Ubuntu 8.04 and 8.10. If someone runs into trouble with a linux distro please let us know.

  1. Choose a link above to start downloading the Fritzing package that's right for you.

  2. Unzip your Fritzing folder somewhere convenient on your hard drive.
    This may also be a good time for you to create a shortcut to the Fritzing application. (A free unzipping program can be found here, should you need it.)

  3. Double click on the Fritzing application, to start Fritzing.
